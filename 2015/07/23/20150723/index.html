大致思路：

1.连接URL。
2.从连接处建立读入流。
3.决定正则表达式
4.对正则表达式编译封装成Pattern。
5.建立Matcher机。
6.find一下，group一下。
接下来的思路，一般是把网页中的URL存储到queue中，从队列中取出，再次爬虫（宽搜）。

import java.net.*;
import java.io.*;
import java.util.regex.*;
public class Spider {
    public static void main(String args[]) throws Exception{
        getMail();
    }
    public static void getMail() throws Exception{
        URL url = new URL("http://tieba.baidu.com/p/3856188322");
        URLConnection con = url.openConnection();
        BufferedReader bufin = new BufferedReader(new InputStreamReader(con.getInputStream()));
        String line = null;
        String mailreg = "\\w+@\\w+[\\.\\w+]+";
        Pattern p = Pattern.compile(mailreg);
        while( (line = bufin.readLine()) != null) {
            Matcher m = p.matcher(line);
            while(m.find()){
                System.out.println(m.group());
            }
        }
    }
}
